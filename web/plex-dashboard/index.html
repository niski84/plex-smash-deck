              OMDb (optional ratings)

              When enabled, Discovery uses the average of TMDB vote average and OMDb IMDb score (both on a 0–10 style scale) for sorting, filtering, and table display. That blends two communities’ ratings and can track closer to how you personally weight critic vs crowd scores. Requires a free API key from OMDb. Responses are cached on disk under data/omdb-cache/ (cleared with Clear TMDB cache on the Discovery tab).


              Sent as apikey= on each request (see omdbapi.com).

              Connectivity

              Playback video goes directly from Plex to the TV; this app does not proxy the movie stream. Background checks run about every 45s (internet, Plex /identity, TMDB, LG SSAP). Every ~4 minutes the server also pulls a small ranged sample (up to 512 KB) from one small library file to estimate Plex throughput. The Dashboard library hint caches Plex title count for 90s to limit duplicate API calls. LG volume polls every 15s (TV only, not Plex). Last update: —

              Connectivity history

              Each time this browser loads fresh data from /api/connectivity, a compact sample is saved locally (same machine as this tab). Use the chart to spot trends. Data stays in your browser — not on the server.
